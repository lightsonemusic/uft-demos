### 2. Frequency Converter
**File:** `2-frequency-converter.html`

Input any frequency → instantly see which proto-word it's closest to.

**Features:**
- Input any frequency (Hz)
- Automatic octave normalization (maps to 210.42–420.84 Hz range)
- Shows distance to nearest proto-word
- Play the input frequency in real-time
- Reference table of all 9 proto-words

**Use Cases:**
- Explore what proto-word a particular frequency corresponds to
- Convert frequencies from different tuning systems
- Educational reference

## How to Use

1. **Open any HTML file in a web browser** (no dependencies, no installation)
2. **Interact with the toys** (click buttons, input values, play sounds)
3. **No sensitive data** — all frequencies and proto-words are public reference material

- All math is public domain (golden ratio, Fibonacci, frequency mapping)
- No external dependencies
